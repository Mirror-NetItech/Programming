Terminologies/Keywords Related To Programming.

*Compiler*:

Is a program that converts a statement written in a particular programming language into machine language before the processor executing it.

*Interpreter*:

Is a program that directly executes instructions written in scripting language without previously compiling them into a machine language.

*Debugging*:

Is the routine process of detecting and removing computer program bugs, errors which is methodically handled by software programmers via debugging tools.

*Bug*:

Is an error, fault in a computer program that causes the program to produce an unexpected result.

*Syntax*:

Is a set of rules that defines the combinations of symbols that are considered to be a correctly structured document in that programming language.

*Assemblers*:

Are translators that are used for translating the program codes written in assembly language into machine language.
